# PasswordGenerator:snake:
Simple python3 password generator.

### USAGE:
```
git clone https://github.com/Keyj33k/PasswordGenerator.git
```
- Now you need to navigate to the Hatake directory and run the password generator:
```
python3 passgen.py
```
