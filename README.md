# PasswordGenerator
Simple python3 password generator.

### Usage:
```
git clone https://github.com/Keyj33k/PasswordGenerator.git
```
- Change current directory:
```
cd PasswordGenerator
```
- Run the code:
```
python3 passgen.py
```
